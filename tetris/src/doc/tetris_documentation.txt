# Документация к игре Tetris

## Клавиши управления игрой
Ниже приведено описание клавиш, используемых для управления игрой Tetris:

- Пробел (Space):
  - Если игра в состоянии STATE_START или STATE_GAME_OVER, начинает новую игру.
  - Если игра активна, вызывает мгновенное падение текущей фигуры (Drop).
- P (латинская буква):
  - Переключает игру между состояниями STATE_MOVING и STATE_PAUSE (Pause).
- Q (латинская буква):
  - Завершает игру (Terminate).
- Стрелка влево (Left):
  - Двигает текущую фигуру влево (Left), если это возможно.
  - Поддерживает удержание для повторного движения с задержкой 100 мс (MOVE_DELAY).
- Стрелка вправо (Right):
  - Двигает текущую фигуру вправо (Right), если это возможно.
  - Поддерживает удержание для повторного движения.
- Стрелка вниз (Down):
  - Ускоряет падение фигуры (Down), если это возможно.
  - Поддерживает удержание для повторного движения.
- Стрелка вверх (Up):
  - Выполняет вращение текущей фигуры (Action).
- W (латинская буква):
  - Альтернативная клавиша для вращения фигуры (Action).
- S (латинская буква):
  - Альтернативная клавиша для ускорения падения (Down).
- A (латинская буква):
  - Альтернативная клавиша для движения влево (Left).
- D (латинская буква):
  - Альтернативная клавиша для движения вправо (Right).

## Команды Makefile
Ниже описаны доступные команды для сборки, установки и управления проектом с использованием Makefile:

- all:
  - Сборка проекта: компилирует исходные файлы tetris.c и cli.c с использованием библиотеки ncurses и создаёт исполняемый файл brickgame.
- install:
  - Устанавливает игру: создаёт папку ../BrickGame/tetris, копирует brickgame в эту папку и создаёт файл high_score.txt для хранения рекордов.
- uninstall:
  - Удаляет игру: удаляет только файл brickgame из ../BrickGame/tetris, оставляя папки и файл рекордов.
- clean:
  - Очищает проект: удаляет объектные файлы (*.o), исполняемый файл brickgame и файл high_score.txt из текущей директории.
- dvi:
  - Показывает документацию: выводит сообщение с указанием открыть этот файл (tetris_documentation.txt) вручную.
- dist:
  - Создаёт дистрибутив: архивирует проект в файл ../brickgame.tar.gz, включая все исходные файлы.
